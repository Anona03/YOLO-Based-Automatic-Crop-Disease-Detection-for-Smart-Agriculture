# YOLO-Based-Automatic-Crop-Disease-Detection-for-Smart-Agriculture
YOLOv8-based crop disease detection system using PlantVillage dataset. Real-time inference (88 FPS on GPU, 32 FPS on edge devices) with novel agricultural metrics: Severity Score Error Rate (SSER) and Economic Impact Score (EIS). Achieves 68.4% mAP@0.5. Built for UAV/field robot deployment.
